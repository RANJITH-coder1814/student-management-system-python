# student-management-system-python
A simple Student Management System built using Python Object-Oriented Programming (OOP) concepts. This project allows users to add, view, and search student records through a menu-driven console interface. 

A simple console-based Student Management System built using Python and Object-Oriented Programming concepts.

## Features
- Add student details
- View all students
- Search student by ID
- Menu-driven program

## OOP Concepts Used
- Class & Objects
- Encapsulation
- Constructors
- Methods

## How to Run
```bash
python student_management.py
